# Usage
please clone Firefox OS gaia and this repository, plug your Firefox OS dev phone
and install it with commands:

```shell
$ cd <GAIA_DIR>
$ MOZILLA_OFFICIAL=1 GAIA_DISTRIBUTION_DIR=<GODZILLA_CUSTOM_PACK_DIR> make production
```
